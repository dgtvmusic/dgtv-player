# DG TV Player

Player On Demand personalizzato per DG TV Music Live Radio.

## Come aggiornare i programmi

- Carica gli MP3 nella cartella `audio/` con lo stesso nome indicato in `data/programs.json`.
- Carica o sostituisci le copertine nella cartella `images/`.
- Modifica `data/programs.json` per cambiare titoli, speaker, descrizioni e file audio.

Nota: gli MP3 inclusi ora sono solo segnaposto demo. Sostituiscili con i file reali.
